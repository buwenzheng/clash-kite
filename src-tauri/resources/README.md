## Resources

This directory is populated automatically by `scripts/prepare.mjs`.

Run `npm install` (or `node scripts/prepare.mjs`) to download:

- `sidecar/mihomo(.exe)` — ClashMeta core binary
- `files/country.mmdb` — IP geolocation database
- `files/geoip.dat` — GeoIP rule data
- `files/geoip.metadb` — GeoIP meta database
- `files/geosite.dat` — GeoSite rule data

These files are git-ignored and not committed to the repository.
