# ClashMeta Binary

This directory should contain the ClashMeta (mihomo) binary file.

## Download

Download the latest mihomo binary from:

- GitHub: https://github.com/MetaCubeX/mihomo/releases

## File Placement

Place the binary file in this directory:

### Windows

```
src-tauri/bin/mihomo.exe
```

### macOS / Linux

```
src-tauri/bin/mihomo
```

## Make Executable (macOS/Linux)

```bash
chmod +x src-tauri/bin/mihomo
```

## Configuration

The application will automatically use this binary to start the proxy service.
